Assignments

Assignment 1 - Basic HTML

Assignment 1 involved creating a webpage using the basic HTML knowledge which I learned during the first week of class. Requirements included a title tag, an h1 header, two four-sentence paragraphs created using the p element, two images which fit appropriately on the page, an unordered list with at least 3 list items (bullet points), a 2 by 3 table, and two external links.

Assignment 2 - Basic CSS

Assignment 2 involved creating a stylesheet and linking it to the index, in order to create a styled webpage. Requirements included a navigation with four links, two paragraphs, two pictures, and a 3 by five table. Styling requirements included changing the font, background, color scheme, and formatting to look more appealing, with elements, ids, and class names each having to be used at least once in the stylesheet.

Assignment 3 - Page Layout

Assignment 3 involved using HTML5 elements to create a webpage with a header, content, and footer. Additionally, Flexbox had to be utilized in the style sheet in order to create a webpage which followed the styling of the example page provided in both landscape and portrait formats.

Assignment 5 - JavaScript

Assignment 5 involved using introductory JavaScript knowledge to create 3 interactive elements, modifying text, element color, and image source.

Assignment 6 - JavaScript Conditionals

Assignment 6 involved using Javascript to create a styled webpage with the following elements: a toggle menu on small screens with a corresponding dropdown arrow and two javascript examples (with only one showing at a time and the other disappearing from the screen). The first example includes a textbox, which displays images when the words bike, scooter, car, and skateboard are typed, and doesn't display an image otherwise. The other example includes 3 buttons labeled "red","green", and "blue" respectively, and a white heart. When one of the buttons are clicked, the heart changes to the corresponding color.

Assignment 7 - JavaScript Loops

Assignment 7 involves the use of loops in javascript order to create a webpage with the following: A button which says "draw stairs", which, when clicked, creates a "staircase" with 10 treads. Additionally, the following appear when "draw stairs" is clicked: a stick figure at the bottom of the stairs, and a "climb stairs" button. When this new button is clicked, the stick figure slowly climbs the staircase, stopping when it reaches the top.

Assignment 8 - JavaScript Arrays

Assignment 8 involes the use of arrays to display a list of various activities when the user loads the page. When clicking one of these activities, a popup showing a corresponding image is displayed. On this pop-up, there is an X button which, when clicked, closes the popup.

Assignment 9 - JavaScript Classes

Assignment 9 involves creating a class with at least 6 attributes, and displaying at least 5 instances of the class using an array. When a user clicks one of the attributes of a respective class instance, a pop-up showing the other attributes of the class, and an animation plays.
